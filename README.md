<h1>Video Game Inventory Application</h1>
<p>VG Inventory Application is a web inventory application that stores an inventory of video games!</p>

<a href='https://vg-inventory-app.onrender.com/'>Live Demo</a>

<h2>How to use VG Inv App (Using live demo above)</h2>
<h3>Searching for a video game</h3>
<p>Video Games are seperated in 3 categories:</p>
<ul>
  <li>
    Games
  </li>
  <li>
    Genres
  </li>
  <li>
    Developers
  </li>
</ul>

<h4>Search By Game</h4>
<ol>
  <li>
    Click on the big "Games" link in the top middle of the page.
  </li>
  <li>
    This will bring up all the video games available on the database.
  </li>
  <li>
    Search for the game you like by scrolling down!
  </li>
</ol>

<h4>Search By Genre</h4>
<ol>
  <li>
    Click on the big "Genres" link in the top middle of the page.
  </li>
  <li>
    This will bring up all the genres available on the database.
  </li>
  <li>
    Click on the desired genre that your desired game has. Ex: RPG
  </li>
  <li>
    All games that have the genre tag you selected (ex: RPG) will be shown.
  </li>
  <li>
    Scroll down until you find your desired game!
  </li>
</ol>

<h4>Search By Developer</h4>
<ol>
  <li>
    Click on the big "Developers" link in the top middle of the page.
  </li>
  <li>
    This will bring up all the developers available on the database.
  </li>
  <li>
    Click on the desired developer that your desired game has. Ex: Nintendo
  </li>
  <li>
    All games that have the developer tag you selected (ex: Nintendo) will be shown.
  </li>
  <li>
    Scroll down until you find your desired game!
  </li>
</ol>

<h3>Adding a new game, genre, or developer</h3>
<p>You can also add new games, genres, or developers to the application. For the desired item to add, click on the desired item to add from the top menu.
Ex: I want to add a genre, I'm going to click on genre.</p>
<ol>
  <li>
    Click on the big "+ add genre" (or game, developer) button.
  </li>
  <li>
    The form for your desired item to add will come up. Fill out the form.
  </li>
  <li>
    When finished, submit the form.
  </li>
  <li>
    Click on genres (or game, developer) and see that your newly added item is there!
  </li>
</ol>
<p>Note: Adding new genres and developers will allow you to use those developers and genres when adding new games.</p>
<h2>Features</h2>
<ul>
  <li>
    Users can sign up using a username, email, and encrypted password
  </li>
  <li>
    Users have historic data that are saved on their profile
  </li>
  <li>
    Users can see other submissions to the same prompt to see how others went about their prompt
  </li>
</ul>
